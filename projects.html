Projects

PDF Summarization and Multilingual Text to Audio Translation

Developing a project enabling users to upload research papers for automated summarization into audio formats in multiple languages, including English, Gujarati, and Hindi. Leveraging pretrained deep learning models to facilitate this process, ensuring efficient summarization and multilingual Text-to-Audio translation.

Optimizing the Parameters of Different Pharmaceutical Machinery

Engaged in optimizing pharmaceutical machinery parameters, including fluidized bed dryers, high shear mixer granulators, and Glatt coating machines. Utilized data visualization and various optimization techniques to ensure consistent batch times by identifying and implementing optimal parameter values.

Employee Attrition Prediction

Developed a Streamlit-based website for Zydus, enabling the HR department to predict employee tenure. The tool allows HR to filter employee data by department, year, date, and month, ensuring secure access and streamlined analysis.

DMAR (Digital Manufacturing Analysis Report): Generating Min/Max CPP Reports for Different Machinery and Batch Comparison

Developed a Python-based back-end system integrated with HTML and Streamlit to facilitate real-time generation of Min/Max CPP reports for different machinery and batch comparisons. The system collected data directly from the PISystem, allowing users to select specific machines, months, years, phase values, and batch numbers to access comprehensive CPP information within the chosen time frame. Additionally, users could download phase-wise Min/Max CPP reports and perform batch comparisons by selecting multiple batches for the same machine.

Document Assist: Facilitating Efficient Interactions with Our AI Chatbot

We are designing and implementing a pioneering chatbot using large language models for a pharmaceutical firm. This chatbot will enable document upload interactions and deliver responses in English audio, significantly improving user accessibility and experience.

Palm Classification using Different Computer Vision Models

Successfully developed a palm classification project distinguishing between male and female palms using various computer vision models. Achieved peak accuracy employing Resnet50, showcasing proficiency in model selection and implementation for enhanced classification outcomes.

Fake News Detection using Different Statistical Models with and without Clickbait

The study explores using machine learning to detect fake news and clickbait on Twitter, created and used clickbait as input parameter to get better accuracy, showcasing LSTM's strong performance and a hybrid model's creation for improved accuracy.

Recommendation System using Non-Negative Matrix Factorization Technique

This study explores using Non-negative Matrix Factorization (NMF) in recommendation systems to improve accuracy and address collaborative filtering challenges, highlighting its interpretability and potential for future enhancements.
